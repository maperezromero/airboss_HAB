## Acerca del proyecto
![Product Name Screen Shot][product-screenshot]

Airboss es un portal de reserva de vuelos que consulta en tiempo real la disponibilidad de asientos en vuelos de todo el mundo.



## Tecnologías utilizadas

-   **BACKEND** - Node.js
-   **FRONTEND** - React.js
-   **BASE DE DATOS** - MySQL
